# AnaEE_Pipeline

 **master_pipeline Parameters :**
 
-    `$1 : Image_docker_name`

-    `$2 : Blazegraph_namespace`

-    `$3 : Ports  number `

-    `$4 : Container Name One `

-    `$5 : Container Name Two `

-    `$6 : Container Name Three `


Ex :

    ./master_pipeline.sh blz_img  ola_namespace 9999 HOST_1 HOST_2 HOST_3
